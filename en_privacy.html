Privacy Policy

Thank you for using our application (hereinafter referred to as “we”). We are committed to not collecting, storing, or sharing any user personal information. This policy applies to all features provided by this application.

1. Scope

  • QR/Barcode Scanning
  • OCR Text Recognition
  • QR Code Generation

2. Information We Do Not Collect

We do not collect any of the following:

  • Personal identity information such as name, phone number, ID number
  • Device identifiers such as IMEI, MAC address
  • Location information such as GPS data
  • User input or OCR-recognized content
  • Behavioral data or usage logs

3. Feature-Specific Data Handling

  • Scan Feature: All processing is done locally. Camera is only used to scan, and results are not uploaded.
  • OCR Recognition: Image recognition is processed locally. Images and recognized text are not stored or transmitted.
  • QR Code Generation: Input is used solely to generate a QR code. Data is not saved or shared.

4. Permission Usage

Permissions are requested only when needed:

  • Camera access: For scanning or image-based text recognition
  • Storage access: For saving or loading images

We promise all permissions are used transparently and never accessed in the background.

5. No Third-Party Integration

This application does not integrate advertisements, third-party SDKs, or analytics tools.

6. Data Security

We do not create user accounts or identity systems. All data is processed locally and never leaves your device.

7. Protection of Minors

We do not intentionally collect data from minors. If you are underage, please use this application with parental guidance.

8. Policy Updates

If major features change, we will notify users via version update notes and update this privacy policy accordingly.
